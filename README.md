## Overview
An interactive Power BI dashboard for analyzing stroke risk and patient health data.

## Dashboard Features
- Total Patients
- Stroke Cases
- Stroke Rate %
- Hypertension Patients
- Heart Disease Patients
- Smoking Status Distribution
- Stroke Occurrence by Gender
- Average Glucose Level by Region
- Average Risk Score by Age Group
- Physical Activity vs Stroke
- Sleep Hours vs Stroke Risk Score

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization
